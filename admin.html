Quick start

  1. Tweak content in the Site / Video / Apps / PDFs tabs.
  2. Open the Generate tab and click Copy config.
  3. On GitHub, open index.html, click ✏️ Edit, replace everything between <!-- CONFIG-START --> and <!-- CONFIG-END -->, and commit.

🧩 Apps 0

Drag the ☰ handle to reorder. Each card on the page shows the image + name + description only; the underlying link is hidden until the visitor clicks.

🧰 Frameworks & Tools 0

External resources (Google Docs, Sheets, etc.) that open in a new tab after the email gate. Use the Drive thumbnail URL https://drive.google.com/thumbnail?id=DOC_ID&sz=w800 for cover images.

📚 Books 0

Published books. Leave the Image URL blank for a styled fallback cover that shows the title, subtitle, and author. The Tone field changes the fallback gradient.

📄 PDFs 0

PDFs render in a custom viewer with downloads disabled. Host the .pdf alongside index.html on GitHub or any CORS-friendly host.
